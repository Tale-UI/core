Accessibility

Built-in accessibility features for inclusive design.

FeatureDetails
.sr-onlyVisually hidden but accessible to screen readers. Uses clip-rect pattern with !important to ensure it always works.
Reduced Motion@media (prefers-reduced-motion: reduce) — Disables all animations and transitions globally when user prefers reduced motion.
