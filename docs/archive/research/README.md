# Archived research

This directory preserves historical research and planning material that informed Tale UI's AI
documentation, component architecture, and agentic UI work.

The documents are retained for context only. They are not normative project documentation and may
describe superseded APIs, plans, or repository structure. Refer to the main documentation and
`CLAUDE.md` for current guidance.

- `analysis/` contains Tale UI and Untitled UI Pro comparative analysis.
- `source-notes/` contains background research and early implementation plans.
